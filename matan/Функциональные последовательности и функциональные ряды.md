### –¢–µ–æ—Ä–µ–º–∞: –∫—Ä–∏—Ç–µ—Ä–∏–π —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ–π —Å—Ö–æ–¥–∏–º–æ—Å—Ç–∏

**–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ**

$\impliedby$. –ò–º–µ–µ–º, —á—Ç–æ $\forall\epsilon > 0 \exists N = N(\epsilon): \forall n > N\ \sup\limits_{x\in X} |f_n(x) - f(x)| < \epsilon$. –ï—Å–ª–∏ $\sup\limits_{x\in X} |f_n(x) - f(x)| < \epsilon$, —Ç–æ –æ—á–µ–≤–∏–¥–Ω–æ, –∏ $\forall x \in X\ |f_n(x) - f(x)| < \epsilon$

### –¢–µ–æ—Ä–µ–º–∞: –∫—Ä–∏—Ç–µ—Ä–∏–π –ö–æ—à–∏ —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ–π —Å—Ö–æ–¥–∏–º–æ—Å—Ç–∏

**–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ**

$\impliedby$. –ò–º–µ–µ–º $$\forall\epsilon > 0\ \exists N = N(\epsilon): \forall x \in X\ \forall n, m > N\ |f_n(x) - f_m(x)| < \epsilon$
–ó–∞—Ñ–∏–∫—Å–∏—Ä—É–µ–º —Ç–æ—á–∫—É $x$. –¢–æ–≥–¥–∞ –ø–æ –∫—Ä–∏—Ç–µ—Ä–∏—é –ö–æ—à–∏ –¥–ª—è –ø—Ä–µ–¥–µ–ª–∞ —Ñ—É–Ω–∫—Ü–∏–∏ –≤ –∫–∞–∂–¥–æ–π —Ç–æ—á–∫–µ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç $\lim\limits_{n\to +\infty} f_n(x) = f(x)$. –í–æ–∑—å–º—ë–º $\epsilon = \dfrac{\epsilon'}{2}$$–ó–Ω–∞–µ–º, —á—Ç–æ $$\forall x \ \in X:\ \exists n, m > N\ |f_n(x) - f_m(x)| < \dfrac{\epsilon'}{2}$$–ó–∞—Ñ–∏–∫—Å–∏—Ä—É–µ–º $n > N$. –¢–æ–≥–¥–∞ $$\forall m > N:\ \forall x \in X\ |f_n(x) - f_m(x)| < \dfrac{\epsilon'}{2}$$–ü–µ—Ä–µ–π–¥—ë–º –∫ –ø—Ä–µ–¥–µ–ª—É –ø–æ $m$: $$\forall x \in X\ \lim\limits_{m\to +\infty}|f_n(x) - f_m(x)| = |f_n(x) - f(x)| \leq \dfrac{\epsilon'}{2} < \epsilon'$$–¢–æ–≥–¥–∞ –ø–æ–ª—É—á–∏–º: $$\forall\epsilon' > 0\ \exists N = N(\epsilon'): \forall x \in X\ \forall n > N\ \ |f_n(x) - f(x)| < \epsilon'$$

## –ü—Ä–∏–∑–Ω–∞–∫–∏ —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ–π —Å—Ö–æ–¥–∏–º–æ—Å—Ç–∏ —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω—ã—Ö —Ä—è–¥–æ–≤

### –¢–µ–æ—Ä–µ–º–∞: –ø—Ä–∏–∑–Ω–∞–∫ –í–µ–π–µ—Ä—à—Ç—Ä–∞—Å—Å–∞
**–¢–µ–æ—Ä–µ–º–∞**

–ï—Å–ª–∏ —Ä—è–¥ $\sum\limits_{k=1}^{+\infty}b_k(x)$ —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ —Å—Ö–æ–¥–∏—Ç—Å—è –Ω–∞ $X$, —Ç–æ $\sum\limits_{k=1}^{+\infty}a_k(x)$ —Ç–æ–∂–µ —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ —Å—Ö–æ–¥–∏—Ç—Å—è –Ω–∞ $X$.

**–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ**
–ü–æ –∫—Ä–∏—Ç–µ—Ä–∏—é –ö–æ—à–∏: $\forall \epsilon > 0 \exists N = N(\epsilon): \forall x \in X\ \forall n > m > N:\ \sum\limits_{k = m+1}^n b_k(x) < \epsilon$.
–¢–æ–≥–¥–∞ –∏–º–µ–µ–º: $$\left| \sum\limits_{k = m + 1}^n\right| \leq \sum\limits_{k = m + 1}^n |a_k(x)| \leq \sum\limits_{k = m+1}^n b_k(x) < \epsilon$$
–¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, —É–ª–æ–≤–∏–µ –ö–æ—à–∏ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –∏ –¥–ª—è $\sum\limits_{k=1}^{+\infty} a_k(x)$. –¢–µ–æ—Ä–µ–º–∞ –¥–æ–∫–∞–∑–∞–Ω–∞.

### –¢–µ–æ—Ä–µ–º–∞: –ø—Ä–∏–∑–Ω–∞–∫ –î–∏—Ä–∏—Ö–ª–µ
**–¢–µ–æ—Ä–µ–º–∞**
–ï—Å–ª–∏
- –ü—Ä–∏ –∫–∞–∂–¥–æ–º —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω–æ–º $x_0 \in X$ —á–∏—Å–ª–æ–≤–∞—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å $a_k(x_0)$ –º–æ–Ω–æ—Ç–æ–Ω–Ω–∞
- –§—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω–∞—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å $a_k(x)$ —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ —Å—Ö–æ–¥–∏—Ç—Å—è –∫ $f \equiv 0$ –Ω–∞ $X$

–¢–æ–≥–¥–∞ —Ä—è–¥ $\sum\limits_{k=1}^{+\infty} a_k(x) b_k(x)$ —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ —Å—Ö–æ–¥–∏—Ç—Å—è –Ω–∞ $X$.

**–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ**
–•–æ—Ç–∏–º –ø–æ–ª—É—á–∏—Ç—å –æ—Ü–µ–Ω–∫—É –≤–∏–¥–∞ $\sum\limits_{k=m+1}^n a_k(x) b_k(x) < \epsilon$, —á—Ç–æ–±—ã —Å–æ—Å–ª–∞—Ç—å—Å—è –Ω–∞ –∫—Ä–∏—Ç–µ—Ä–∏–π –ö–æ—à–∏. –ü—Ä–∏–º–µ–Ω—è—è —Ç–µ –∂–µ —Ä–∞—Å—Å—É–∂–¥–µ–Ω–∏—è, —á—Ç–æ –∏ –≤ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–∏ –ê–±–µ–ª—è –ø–æ–ª—É—á–∞–µ–º $$\sum\limits_{k = m+1} a_k(x) b_k(x) = a_n(x) V_n(x) - \sum\limits_{k = m+1} V_k(x)(a_{k+1}(x) - a_k(x))$$ –ó–¥–µ—Å—å $V_j(x) = \sum\limits_{k = m+1}^j b_k(x),\ V_m(x) := 0$. –í—Å—ë –∞–Ω–∞–ª–æ–≥–∏—á–Ω–æ –æ–±—ã—á–Ω—Ç–æ–º—É –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—é –ê–±–µ–ª—è, –≥–¥–µ —Å–¥–≤–∏–Ω—É—Ç –Ω–∏–∂–Ω–∏–π –∏–Ω–¥–µ–∫—Å.

–ü–æ —É—Å–ª–æ–≤–∏—é —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ–π –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–Ω–æ—Å—Ç–∏:
$$|V_j(x)| = \left|\sum\limits_{k = m+1}^j b_k(x)\right| = \left|\sum\limits_{k=1}^j b_k(x) - \sum\limits_{k=1}^m b_k(x) \right| \leq$$ $$\leq \left|\sum\limits_{k=1}^j b_k(x)\right|+\left|\sum\limits_{k=1}^{m} b_k(x)\right| \leq 2M$$
–ü–æ–ª—É—á–∞–µ–º $$\left|\sum\limits_{k = m+1}^n a_k(x)b_k(x)\right| = \left|a_n(x)V_n(x) - \sum\limits_{k = m+1}^{n-1} V_k(x)(a_{k+1}(x) - a_k(x))\right| \leq$$ $$\leq |a_n(x) V_n(x)| + \left|\sum\limits_{k = m+1}^{n-1} V_k(x)(a_{k+1}(x) - a_k(x))\right| \leq$$ $$\leq 2M|a_n(x)| + 2M\sum\limits_{k = m+1}^{n - 1} |a_{k+1}(x) - a_k(x)|\quad (ü¶ã)$$
